# Course Registration System (Java)

This project is a console-based Course Registration System developed using Java. 
It allows students to create accounts, explore courses, register for courses, and simulate payment. 
The system stores user and course registration data using file handling.

## Features

- User Registration
- User Login System
- Course Listing
- Course Details (Duration, Fees, Description)
- Course Registration
- Fake Payment Simulation
- Student Dashboard
- View Registered Courses
- Admin Panel
- Receipt Generation
- File Handling for Data Storage

## Technologies Used

- Java
- File Handling
- Console-based Interface

## Project Structure

RegistrationSystem.java – Main program file  
users.txt – Stores registered users  
course_reg.txt – Stores course registrations  
receipt.txt – Generated after payment  

## How to Run

1. Open terminal in the project folder
2. Compile the program
   javac RegistrationSystem.java
3. Run the program
   java RegistrationSystem
